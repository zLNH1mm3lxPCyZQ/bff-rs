# Boundary First Flattening (BFF)

A Rust implementation of [Boundary First Flattening](https://arxiv.org/abs/1704.06873) for conformal surface flattening.

## Motivation

This port exists to make a valuable algorithm easier to use across platforms.

## References

- **Paper:** Sawhney & Crane, *Boundary First Flattening* ([arXiv:1704.06873](https://arxiv.org/abs/1704.06873))
- **Original implementation:** [https://github.com/GeometryCollective/boundary-first-flattening](https://github.com/GeometryCollective/boundary-first-flattening)
